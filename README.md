# Team Task Manager

A full-stack Team Task Management application built using the MERN stack for managing projects, assigning tasks, tracking progress, and generating reports.

## Features

- User Authentication (Login/Register)
- Admin and Member roles
- Project creation and management
- Assign projects to members
- Task assignment and tracking
- Task status updates
- Dashboard statistics
- Reports with Pie Charts and Bar Charts
- Role-based access control
- Responsive UI

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Recharts
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Run Locally

### Frontend

```bash
cd client
npm install
npm run dev
```

Frontend runs on:

```bash
https://team-task-manager-jade-ten.vercel.app/
```

### Backend

```bash
cd server
npm install
npm start
```

Backend runs on:

```bash
https://team-task-manager-production-1add0.up.railway.app/
```
